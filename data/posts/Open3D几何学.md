### 点云
**Visualize point cloud 可视化点云** 是三维数据处理的起点。它把抽象的点云数据转换成可交互的 3D 场景，让我们能像拿放大镜一样观察物体的空间结构、密度分布和采集缺陷。点云可视化不仅用于调试算法，也广泛用于虚拟现实、逆向建模和空间分析中。

**Voxel downsampling 体素下采样** 是通过把空间划分为均匀小立方体（体素），用每个体素内的一个代表点（如中心或平均点）代替原始所有点。它本质上是一种空间量化，既能减少冗余、压缩数据，又能保持几何形状，是点云预处理中的常规步骤。在移动设备、嵌入式视觉和实时渲染中尤为重要。

**Vertex normal estimation 顶点法线估计** 是计算点云中每个点的“朝向”。它是通过分析每个点邻域的主方向（例如 PCA）来估算法线向量，类似于判断一个表面“朝哪边”。法线用于三角网格重建、表面光照模拟、形状理解等，在视觉几何和渲染中是基础要素。

**Access estimated vertex normal 访问估计的顶点法线** 是程序获取前面估算的法线向量，用于判断点之间的朝向一致性、边缘识别、或者导出数据到其他系统（如 Blender）。它为点云提供了方向信息，使得点不再是“无向的雾”。

**Crop point cloud 裁剪点云** 是通过空间选框或规则几何体，从原始点云中提取出感兴趣区域，比如框选一个桌子、某个机械零件或人体部分。裁剪对于构建局部模型、去除无关背景或实现“聚焦处理”极为关键，是点云处理的局部化入口。

**Paint point cloud 绘制点云** 不仅是美观，它还能通过颜色表达信息，比如用红色表示温度高、蓝色表示误差大，或用分类标签着色。上色后的点云更具表达力，在检测、分割、交互分析等场景下可极大增强可读性。

**Point cloud distance 点云距离** 是计算两个点云之间的最近点距离，常用于误差评估、变化检测或点云匹配精度测量。例如在机器人建图中，比较当前扫描与地图点云的距离可以判断定位误差；在重建质量评估中，用 GT（真实模型）和重建点云的距离衡量模型精度。

**Bounding volumes 包围体** 是构建一个包含整个点云的最小几何边界，比如 AABB（轴对齐包围盒）或球体。这一过程常用于碰撞检测、场景裁剪、空间索引或粗粒度筛选，是点云空间感知与加速的基础技术。

**Convex hull 凸包** 是数学中最小凸封闭空间的构造，在点云中应用可提取轮廓、估算体积、生成边界网格。它类似用一张橡皮筋在空间中套住点云最外围的点，是几何形状分析的重要工具。

**<font style="background-color:#FBDE28;">DBSCAN clustering DBSCAN 聚类</font>** 是无监督点云聚类算法，能自动将密度高的点归为一类，噪声点视为离群。它在点云物体分割、障碍物识别、结构识别中应用广泛，尤其适合分割散落在空间中的多个独立物体，比如识别一堆箱子或行人。

 从“密度”这个关键词入手。它不是靠距离最近的点，而是看某个点附近的点够不够多，像“人多的地方就是一群”。只要一个点的 ε 邻域内的点数超过最小阈值 minPts，这个点就成了“核心点”。其他能被核心点直接或间接“带动”的点，会被归进同一类，而那些太稀疏的点被当成噪声。  

**<font style="background-color:#FBDE28;">Plane segmentation 平面分割</font>** 是从点云中提取具有平面结构的区域，典型方法如 RANSAC 拟合平面。这在建筑扫描（如提取地板、墙面、天花板）、桌面物品识别、工业CAD建模中极其重要，是几何理解的第一步。

常用的 RANSAC 算法会不断从点云中随机取三点拟合平面，评估有多少点与这个平面“足够接近”，这些点叫做内点。重复多次后选出最优平面。如果你想找多个平面，通常是找一个平面后把它“拿走”，然后在剩下点云中再找。  

**<font style="background-color:#FBDE28;">Planar patch detection 平面块检测</font>** 是平面分割的精细化，它关注小范围局部平整区域，并可提取出多个小 patch。相比整体平面，它能更好处理曲面中“局部平”，适用于三维拼图建模、家具识别、表面损伤检测等场景。

已分割出的平面进一步细分的一步。虽然一个大平面数学上是完整的，但它可能对应多个“物理结构块”，比如墙面上窗户、门、告示牌等。检测平面块常用的思路是：在大平面上再进行区域生长（region growing），要求区域内部的法向量差异小，边界平滑。  

**<font style="background-color:#FBDE28;">Hidden point removal 隐藏点去除</font>** 是一种基于视角的点云“可见性”处理，常用于模拟实际相机/观察者只能看到的部分。例如从一个视角观察一辆车，车背后就应该不可见，去除这些隐藏点可获得真实感的点云视图，也用于渲染加速和相机遮挡模拟。

模拟“从一个观察点看过去，能看到点云的哪些部分”。一种常用方法是把点云以某个视点为中心，投影到一个球面或用凸包包裹，然后把这些点“拉回去”构建一个可见面。如果一个点在其他点“背后”，就会被视为不可见。这个操作可以加速渲染、减少 occlusion，也常用于生成可见视图的 mesh。

### 三角网络
**Visualize a 3D mesh 可视化 3D 网格** 是三维模型处理的起点，它将由顶点和三角面片构成的网格渲染在窗口中，让我们可以像手握雕塑一样观察模型的形状、边界、结构和平滑度。它支持旋转、缩放、着色，是调试、比对和展示 3D 模型的基础手段。

**Surface normal estimation 表面法线估计** 是计算网格每个顶点的方向信息，像是在每个表面插入一支小箭头，表示它“朝向哪里”。这个方向信息用于光照模拟、网格平滑、边缘检测等，特别关键于渲染时计算高光与阴影，也在很多几何算法中起到决定性作用。

**Crop mesh 裁剪网格** 是指对一个复杂模型只保留我们关心的一部分，例如剪掉背景、边缘或者支撑结构。这就像用虚拟剪刀在三维空间中选择区域、清理噪音点或者聚焦分析某个子结构，是三维建模和重建后的常规处理步骤。

**Paint mesh 绘制网格** 指的是给网格表面添加颜色，让模型更具可视化表达力。它可以统一着色，也可以根据高度、曲率、法线方向或分类标签进行上色，常用于可视对比、语义分割展示、或者将材质编码为颜色进行调试。

**Mesh properties 网格属性** 是对网格整体的结构信息读取与管理，包括顶点数、面数、是否封闭、是否有法线或颜色、是否流形等。这相当于查看网格的“身份证”，掌握它是否可以用于物理模拟、可视化渲染或是否需要修复补洞。

**Mesh filtering 网格过滤** 指的是平滑网格表面，类似 Photoshop 图像的模糊滤波。它可以去除扫描噪声、缩减锯齿、修正抖动面片，让网格更连续。  
**Average filter 平均滤波** 是最简单的方式，每个顶点取周围顶点的平均位置更新，但可能会导致整体模型收缩。  
**Laplacian 拉普拉斯算子** 是基于拓扑结构的滤波器，它考虑顶点与邻接面的关系，让模型表面变得更光滑自然，常用于初步几何平滑。  
**Taubin filter Taubin滤波器** 是改进的拉普拉斯滤波，它通过添加“反向平滑”步骤来抑制收缩，在不失精度的情况下实现柔化，是工业建模中常用的高级滤波方法。

**Sampling 采样** 是指在网格表面均匀（或按策略）采样点，生成点云。它的意义在于将网格转化为点云数据，便于后续特征提取、仿真扫描、点云重建或可视化下采样。

**Mesh subdivision 网格细分** 是增加网格的面片数量，让模型更精细光滑，就像把粗糙多边形“打磨抛光”。它在动画模型、曲面插值和渲染中非常重要，是构建高精度网格的重要手段之一。

**Mesh simplification 网格简化** 是与细分相反的操作：将一个高面数模型压缩为低面数版本，保持几何形状大致不变。这用于加速渲染、压缩存储、提高实时性，尤其适合在游戏、AR/VR 和移动端部署。  
**Vertex clustering 顶点聚类** 是一种快速简化方式，它将空间中接近的点聚为一个点，用于粗糙但高效压缩模型。  
**Mesh decimation 网格抽取** 是保留关键结构的同时，有选择性地删除或合并面片，常用 QEM 算法控制几何误差，是最主流的三角网格简化方法之一。

**Connected components 连通分量** 是分析一个网格中是否存在多个不连接的部分，就像判断一块碎掉的陶瓷中有几块碎片。这在分割模型、修复裂缝、识别独立子物体中非常有用，是复杂模型结构分析的关键步骤。

### Processing
**File IO 文件 IO** 模块支持三类主要数据：点云、网格、图像的读写操作。就像是模型处理的“入口和出口”，你可以加载外部文件，也可以把处理结果导出保存，常用格式如 `.pcd`、`.ply`、`.obj`、`.stl`、`.png` 等。

**Point cloud outlier removal 点云离群点去除** 是清洗原始点云的关键步骤。点云中常包含扫描误差或噪声点，这些像是“空气中的灰尘”一样会影响模型质量。常用方法包括：

+ **Prepare input data 准备输入数据**：预处理步骤，比如归一化、坐标变换；
+ **Select down sample 选择**下采样：减少点数，加速处理；
+ **Statistical outlier removal 统计**离群点去除：基于邻居统计分析判断异常点；
+ **Radius outlier removal 半径异常值去除**：判断点周围半径内邻居数，太少就删除。

**Voxelization 体素化** 是将点云或网格离散为一个个立方小盒子（voxel），就像把模型转换成 Minecraft 风格的体素世界：

+ **From triangle mesh 从三角网格** 和 **From point cloud 从点云** 都是将稠密数据转为稀疏体素；
+ **Inclusion test 包含性测试** 判断某点是否在体素体积内；
+ **Voxel carving 体素雕刻** 用多视角图像反推出对象体素，常用于空间建模和隐空间模拟。

**Octree 八叉树** 是对三维空间分层划分的高效结构，将空间分成多个小立方体并不断细分，用于加速搜索和管理点云/体素：

+ 可以从 **点云** 或 **体素网格** 构建；
+ **Traversal 遍历** 是访问结构的方式；
+ **Find leaf node containing point 查找包含点的叶节点** 是查找一个点在哪个子空间中。

**Surface reconstruction 表面重建** 是从点云生成网格的核心模块，重建一个连续的 3D 表面：

+ **Alpha shapes Alpha 形状** 是一种轮廓提取方法；
+ **Ball pivoting 球体旋转** 模拟一个球在点间“滚动”，连接构建三角面；
+ **Poisson surface reconstruction 泊松表面重建** 是主流算法，能从点云和法线构建平滑模型；
+ **Normal estimation 法线估计** 是前置条件，用于构建方向连续的面。

**Transformation 变换** 包括：

+ **Translate 平移**、**Rotate 旋转**、**Scale 比例**：构成基础三维几何变换；
+ **General transformation 一般变换**：可包含平移旋转缩放齐次矩阵变换，应用于配准、对齐、多视角合成等。

**Mesh deformation 网格变形** 是对已有网格形状进行柔性编辑：

+ **Smoothed ARAP 平滑 ARAP**（As-Rigid-As-Possible）在变形过程中尽量保持原局部刚性结构；
+ **Intrinsic shape signatures (ISS) 内在形状签名** 是识别模型重要结构（如角点、棱角）的一种特征检测方法；
    - **ISS Keypoints ISS 关键点**：提取网格或点云的几何关键点；
    - **ISS keypoint detection example**：检测示例用于语义分割或模型识别。

**Ray Casting 射线投射** 是模拟从视角发出的“光线”穿过三维模型的过程：

+ **Creating a virtual point cloud 创建虚拟点云**：从多角度“扫描”模型生成新点云，常用于模拟激光雷达、深度图生成等。

**Distance Queries 距离查询** 是计算空间中任意两个对象间的最近距离：

+ **Converting a mesh to an implicit representation 将网格转换为隐式表示**：用于分析任意点是否在网格内；
+ **Computing distances with closest point queries 使用最近点查询计算距离**：适合碰撞检测、模型比较等应用。

**UV Maps UV贴图** 是给网格表面“包裹纹理”的方式，就像给 3D 模型披上一张二维皮肤：

+ **What is a UV Map? 什么是 UV 贴图？**：定义如何将二维图像映射到三维表面；
+ **How to add custom UV maps? 如何添加自定义 UV 贴图？**：实现个性化图案、材质投影；
+ **Quick reference to default UV Maps** 是对各种标准几何体的内建 UV 映射，如：
    - **Box（框/盒子）** 是否每个面映射；
    - **Tetrahedral 四面体**、**Octahedral 八面体**、**Icosahedron 二十面体** 等为几何基元；
    - **Cylinder 圆柱体**、**Cone 圆锥体**、**Sphere 球体** 是常见参数化形体。

## 🧭 一、采集点云
**目的**：获取真实场景的三维空间离散点数据  
**常见方法**：

+ 激光雷达（LiDAR）
+ 深度摄像头（如Intel RealSense、Kinect）
+ 结构光/TOF相机
+ SfM/MVS（通过多张图像+相机位姿还原三维点）

---

## 🧹 二、预处理
**目的**：去除噪声，提高点云质量

### 1. **降采样（Downsampling）**
+ 方法：Voxel Grid 下采样
+ 原理：将点云按体素划分，对每个体素仅保留一个点
+ 作用：减小数据规模，提高计算效率

### 2. **去噪（Outlier Removal）**
+ 统计离群值去除：判断某点邻域密度是否过低
+ 半径滤波去除：点的邻域内点数不足则删除

---

## 🎨 三、特征计算
**目的**：为重建模型提供结构信息支撑

### 1. **估计法线（Normal Estimation）**
+ 使用k邻域点拟合平面，估计表面法线方向
+ 可选方向一致性处理（Orient normals consistently）

### 2. **曲率估计（可选）**
+ 通过PCA主成分分析估算表面变化剧烈程度（用于高斯采样等）

---

## 💡 四、点云增强（可选）
**目的**：使点云更密集、边缘更清晰  
**方法**：

+ 高斯泼溅（Gaussian Splatting）：根据法线和曲率密度自适应生成虚拟点
+ 点云融合（如多帧合并）：从多个视角融合生成稠密点云

---

## 🧱 五、表面重建（Surface Reconstruction）
**目的**：将稠密点云转换为连续曲面网格（三角网）

### 常用方法：
#### 1. **泊松重建（Poisson Reconstruction）** ✅最常用
+ 适合带法线的稠密点云
+ 对曲面连续性要求高，效果光滑

#### 2. **Alpha Shape（α形状）**
+ 适合稀疏点云和封闭体提取
+ 基于 Delaunay 三角剖分

#### 3. **Ball Pivoting（球体旋转）**
+ 用虚拟小球滚动在点云上“描边”，形成三角面片
+ 对于扫描数据效果较好，但对密度敏感

---

## ✂️ 六、网格后处理
**目的**：优化网格质量，提高视觉和物理仿真效果

### 1. **去除低质量面片**
+ 如密度过低区域、孤立面等

### 2. **网格简化（Simplification）**
+ 减少面片数量但保留形状特征（如Quadric Error Simplification）

### 3. **网格平滑（Smoothing）**
+ 拉普拉斯平滑、Taubin滤波等方式消除噪声但保留细节

---

## 👁 七、可视化与导出
+ 使用Open3D、MeshLab或Blender查看重建效果
+ 导出为 `.ply`, `.obj`, `.stl`, `.glb` 等标准格式用于3D打印、仿真或可视化系统

---

## ✅ 总结流程图：
```plain
采集点云 → 点云预处理（降采样+去噪） → 法线估计 → 稠密增强（如高斯泼溅） 
→ 表面重建（泊松/球体旋转） → 网格优化（平滑+简化） → 可视化&导出
```
